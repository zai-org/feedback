# 我该去哪儿？ · Where to Ask

## 快速分流 · Quick Routing

| 你的情况 / Your case | 去哪里 / Where | 期望响应 / SLA |
| --- | --- | --- |
| 💡 **想要新功能** / Want a feature | [新建 Issue（功能建议）](../../issues/new?template=feature_request.yml) | 3 工作日内首次响应 |
| 🐛 **遇到 Bug** / Found a bug | [新建 Issue（Bug）](../../issues/new?template=bug_report.yml) | 3 工作日内首次响应 |
| 🙋 **不知道怎么用** / How do I…? | [Discussions → Q&A](../../discussions/categories/q-a) | 社区互助，无 SLA |

| 🚨 **紧急生产故障** / Production incident | 邮件 cyunying090@gmail.com 主题 `[ZCode URGENT]` | 尽快处理 |

## 我们不在这里处理的 · Not Handled Here

- 商务合作、采购询价 → 走官方商务渠道
  *Business / sales inquiries → use official sales channels*
- 账号 / 计费 / 密码重置 → 走客服支持
  *Account / billing / password reset → contact customer support*

如不确定，先开个 Discussion，我们会引导你到正确渠道。
When in doubt, start a Discussion — we'll point you to the right place.
